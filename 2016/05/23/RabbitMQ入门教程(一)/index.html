RabbitMQ安装

安装RabbitMQ server

  1. 下载rabbitmq-server安装包

  1. RabbitMQ的编写语言是erlang,因此，安装之前需要先安装erlang

    因为erlang依赖的其他的软件包，需要先安装依赖包

    再重新执行

  2. 验证erlang安装，输入erl，如果成功，会出现

  3. 安装RabbitMQ server

启动RabbitMQ服务

  1. 进入RabbitMQ安装目录

  2. 启动服务

  3. 登录http://127.0.0.1:15672/ 输入默认账号guest和默认密码client
